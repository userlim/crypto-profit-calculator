   npm install

   Open http://localhost:3000

DOCKER DEPLOYMENT
=================

Dockerfile:
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]
```

Build and Run:
```bash
docker build -t crypto-calculator .
docker run -p 3000:3000 crypto-calculator
```

STATIC EXPORT (Optional)
=========================

For static deployment (no server needed):

1. Update next.config.js:
   ```javascript
   const nextConfig = {
     output: 'export',
     reactStrictMode: true,
   };
   module.exports = nextConfig;
   ```

2. Build:
   npm run build

3. Output is in ./out directory
   Can be deployed to S3, Netlify, GitHub Pages, etc.

TROUBLESHOOTING
===============

Issue: Tailwind styles not applying
Solution: Verify tailwind.config.ts content paths are correct

Issue: TypeScript errors
Solution: Run npx tsc --noEmit to check for errors

Issue: Calculator not saving state
Solution: Check localStorage is enabled in browser

Issue: Images not loading
Solution: Verify public folder path is correct

Issue: GA not tracking
Solution: Check GA code in layout.tsx, verify domain in GA
